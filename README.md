# Varun - Motion Graphics Designer Portfolio

A cinematic, interactive portfolio website for a motion graphics designer with premium UI and smooth animations.

## 🎬 Cinematic Intro Features

### Intro Sequence
1. **Dark Screen Start** - Begins with a dark screen with subtle grain texture
2. **Ambient Glow** - Soft animated light glows from top-left and bottom-right
3. **Text Reveal** - Staggered text animation:
   - "VARUN" (fade + upward motion)
   - "Motion Graphics Designer" (fade + blur)
   - Tagline: "I create visuals that move, engage, and convert."
4. **Smooth Transition** - After 3.5 seconds, seamlessly transitions to main UI

### Animation Style
- **Smooth cinematic transitions** (no fast/harsh motion)
- **Fade + blur + slight zoom** effects
- **Staggered timing** for text appearance
- **Apple-style product reveal** feel

## 🎨 Interactive UI Features

### Navigation System
- **Panel-based navigation** (left sidebar)
- **Floating glass cards** with hover effects
- **Active state indicators** with gradient highlights
- **Smooth section transitions**

### Visual Design
- **Dark premium gradient** (#0a0a0a to #111111)
- **Glassmorphism cards** with backdrop blur
- **Neon blue/purple accents**
- **Rounded corners** (20px)
- **Soft shadows and depth**

### Micro-interactions
- **Custom cursor** with glow effect
- **Hover animations**:
  - Scale (1.04)
  - Lift (translateY -8px)
  - Glow effect
  - Smooth transition (0.3s ease)
- **Click animations**:
  - Scale (0.95 to 1)
  - Background blur
  - Smooth transition (0.4-0.5s)

### Motion System
- **Idle floating animation** on cards
- **Subtle gradient movement** in background
- **Parallax-like depth effects**
- **Staggered text appearances**

## 🎯 Design Principles

### Mood
- **Cinematic** - Like a film opening
- **Premium** - High-end product feel
- **Calm** - No flashy or fast animations
- **Minimal** - Clean, uncluttered interface

### Avoided
- Flashy effects
- Fast animations
- Overcomplicated navigation
- Gaming-heavy aesthetics
